PC Building Guide

Video Tutorial

Complete guide to assembling and disassembling a PC

Diagram

PC components layout – CPU, RAM, GPU, PSU, storage
Required Tools & Anti-static
  • Phillips screwdriver (#2), anti-static wrist strap, thermal paste, zip ties.
  • Work on a non-carpeted surface, touch metal case frequently.
CPU & Cooler Installation
  1. Open CPU socket lever, align gold triangle, place CPU gently.
  2. Apply pea-sized thermal paste.
  3. Mount cooler with correct orientation, tighten evenly.
RAM & Storage
  • Install RAM in slots A2/B2 for dual-channel.
  • Mount SSD in M.2 slot or 2.5" bay; HDD in 3.5" bay.
GPU & PSU
  • Insert GPU into PCIe x16 slot, secure with screws.
  • Connect 24-pin motherboard and 8-pin CPU power.
  • Route cables neatly; use cable ties.
Front Panel & BIOS
  • Connect power switch, reset, HDD LED, USB, audio.
  • First boot: enter BIOS (Del/F2) to set boot order.
  • Common mistakes: forgetting CPU power, incorrect RAM slots.

Ethernet Installation

Ethernet standards: CAT5e (1 Gbps), CAT6 (10 Gbps up to 55m), CAT6A (10 Gbps 100m), CAT7 (shielded).

Video Tutorial

How to crimp RJ45 connectors – straight-through and crossover

Wiring Diagram

T568A vs T568B – straight-through (same ends) and crossover (one end each)
RJ45 Connectors & Wiring

T568A: White/Green, Green, White/Orange, Blue, White/Blue, Orange, White/Brown, Brown.

T568B: White/Orange, Orange, White/Green, Blue, White/Blue, Green, White/Brown, Brown.

Straight-through: both ends same standard. Crossover: one end T568A, other T568B.

T568B: WO O WG B WB G WBr Br
Crimping Process & Tester
  1. Strip 2.5cm of jacket, untwist pairs, arrange per standard.
  2. Cut wires flush, insert into RJ45, crimp firmly.
  3. Test with cable tester; check continuity and pin mapping.
Common Problems
  • Incorrect wiring order → no link.
  • Untwisted pairs too long → crosstalk.
  • Damaged cable → test with TDR.

Computer Disassembly

Video Tutorial

Complete PC teardown – remove PSU, GPU, RAM, CPU, motherboard

Disassembly Flowchart

Step-by-step disassembly order – power off → remove cables → components
  1. Power off, unplug all cables, press power button to discharge.
  2. Remove side panels, lay case flat.
  3. Disconnect PSU cables from motherboard, GPU, drives.
  4. Remove GPU, RAM, CPU cooler, CPU (in that order).
  5. Unscrew motherboard and remove.
  6. Clean components with compressed air and soft brush.
  7. Organize screws in labeled containers.
  8. Reassembly is reverse process.

BIOS & UEFI

How to navigate and configure BIOS/UEFI settings

BIOS Screen

Typical UEFI BIOS interface – boot order, fan curves, XMP

  • Set boot order, enable XMP for RAM, configure fans.
  • UEFI offers Secure Boot, TPM, and faster boot.
  • Update BIOS via USB (only if needed).

Error Codes Chart

BIOS beep codes – what they mean and how to fix
No Display / No Power

Check power cable, PSU switch, monitor input, reseat GPU/RAM.
